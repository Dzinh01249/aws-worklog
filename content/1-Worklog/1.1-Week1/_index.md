Week 1 Objectives:
- Initialize the project, explore the AWS ecosystem, and set up the basic development environment.

| Date | Tasks | Start Date | End Date | References |
|------|-------|------------|----------|------------|
| 1 | Read the internship process guidelines and watch overview videos introducing the AWS ecosystem. | 17/04/2026 | 17/04/2026 | [AWS Training](https://aws.amazon.com/training/) |
| 2 | Study Cloud Computing definitions, cloud deployment models, and core benefits. | 18/04/2026 | 18/04/2026 | [What is Cloud?](https://aws.amazon.com/what-is-cloud-computing/) |
| 3 | Register an AWS Free Tier account. Practice security: create IAM Admin Group and enable MFA. | 20/04/2026 | 20/04/2026 | [AWS Free Tier](https://aws.amazon.com/free/)<br>[IAM Best Practices](https://docs.aws.amazon.com/IAM/latest/UserGuide/best-practices.html) |
| 4 | Install the Hugo Static Site Generator framework locally and learn repository management. | 21/04/2026 | 21/04/2026 | [Hugo Docs](https://gohugo.io/documentation/) |
| 5 | Learn fundamental theory on Amazon EC2 virtual servers (Instances, Security Groups). | 22/04/2026 | 22/04/2026 | [Amazon EC2](https://aws.amazon.com/ec2/) |
| 6 | Perform basic lab exercises on the AWS Console to get familiar with the UI and Billing. | 23/04/2026 | 23/04/2026 | [AWS Billing](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/billing-what-is.html) |

Week 1 Achievements:

- AWS account securely created.
- Grasped foundational Cloud concepts.
