Week 1 Objectives:
- Get accustomed to the workspace, grasp core cloud computing concepts, and set up the documentation tool (Hugo).

| Date | Tasks | Start Date | End Date | References |
|------|-------|------------|----------|------------|
| 1 | Attend the orientation session, learn about company culture, and watch introductory videos on core AWS services. | 17/04/2026 | 17/04/2026 | [AWS Overview](https://aws.amazon.com/what-is-aws/) |
| 2 | In-depth study on Cloud Computing:<br>- Differentiate IaaS, PaaS, SaaS.<br>- Pros and cons of Public, Private, and Hybrid Cloud. | 18/04/2026 | 18/04/2026 | [Cloud Types](https://aws.amazon.com/types-of-cloud-computing/) |
| 3 | Create an AWS Free Tier account and set up basic security:<br>- Create an IAM User and assign access permissions.<br>- Configure Multi-Factor Authentication (MFA) for the Root account. | 20/04/2026 | 20/04/2026 | [AWS IAM](https://aws.amazon.com/iam/) |
| 4 | Set up the Worklog environment:<br>- Install Git, Go, and Hugo locally.<br>- Initialize the first Hugo project and get familiar with its directory structure. | 21/04/2026 | 21/04/2026 | [Hugo Quickstart](https://gohugo.io/getting-started/quick-start/) |
| 5 | Explore the Amazon EC2 computing service:<br>- Read documentation on Instance families (T3, M5, C5...).<br>- Understand how Security Groups act as virtual firewalls. | 22/04/2026 | 22/04/2026 | [Amazon EC2 Basics](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/concepts.html) |
| 6 | Hands-on experience with the AWS Management Console:<br>- Trial launch of a t2.micro EC2 server (Ubuntu).<br>- Set up a Budget Alert in AWS Billing to control costs. | 23/04/2026 | 23/04/2026 | [AWS Budgets](https://aws.amazon.com/aws-cost-management/aws-budgets/) |

Week 1 Achievements:

- Successfully set up and fully secured a personal AWS account.
- Completed building a static website using the Hugo framework to host the Worklog.
- Gained first hands-on experience by successfully launching an EC2 virtual server.
